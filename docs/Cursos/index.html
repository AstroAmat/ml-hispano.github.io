    Listado de cursos propuestos por los miembros de la comunidad Machine Learning Hispano. Si tu también quieres formar parte de esta comunidad, puedes unirte a través de este link : https://bit.ly/2CI89ne

    Cursos de CS/ML/DL¶

    • Curso introductorio impartido por DotCSV : Curso de "Introducción al aprendizaje automático" que impartió DotCSV en noviembre de 2018. Los ejercicios van desde modelos simples, como la regresión lineal hasta técnicas más avanzadas como las redes neuronales convolucionales.
    • Open Source Society University : Curriculum universitario completo en Informática gratis sin certificación.
    • Coursera Machine Learning : Curso introductorio de Machine Leaerning de Coursera impartido por Andrew Ng.
    • Scrimba Neural Networks in Javascript : Aprende a construir tu Red Neuronal en Javascript con la librerían Brain.js.
    • Udacity Intro to Deep Learning with PyTorch : Curso introductorio de Deep Learning con PyTorch en Udacity.
    • Stratio Tutorial de Deep Learning P1 | P2 | P3 : Talleres introductorios de Deep Learning.
    • Stanford CS229 - Machine Learning : Completo curso de Machine Learning.
    • Técnico en Minería de Datos : Plan de capacitación en minería de datos.
    • Fast.AI Practical Deep Learning For Coders, P1 | P2 : Curso de Deep Learning, impartido por Jeremy Howard.
    • Curso intensivo de aprendizaje automático |: Curso intensivo de aprendizaje automático de Google.
    • Kadenze Creative Applications of Deep Learning with TensorFlow : Creación de apps creativas con Deep Learning.
    • CalTech Learning from data : Curso de Machine Learning de la Universidad de California.
    • Data Science with Python : Curso de Data Science impartido por IBM.
    • Elements of AI : Curso introductorio de Machine Learning de la Universidad de Helsinki.
    • Iniciación a IA : Curso de inicio en IA basado en el videojuego Pacman con proyectos para aprender y programar en Python.
    • Viendo la teoría : Una introducción visual a probabilidad y estadística.
    • Curso de estadística computacional : Curso de estadística computacional online.
    • Introducción a Machine Learning por la Agencia Espacial Europea : En 2018, la Agencia Espacial Europea organizó un curso de introducción al Machine Learning. El curso habla de conceptos de ML y técnicas tales como regresión lineal, polinómica, regresión logística, máquinas de vectores de soporte, árboles de decisión, bosques aleatorios (random forests), redes neuronales profundas (deep learning), aprendizaje no-supervisado y procesamiento del lenguaje natural. Incluye teoría y ejemplos prácticos en python.
